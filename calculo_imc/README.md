## ℹ️ Dados do Sistema
### Fórmula 
$$IMC = \frac{Peso (kg)}{Altura (m)^2}$$

Este é o cálculo matemático utilizado para rodar o sistema.
___

## 🚀 Como rodar o projeto

1. Certifique-se de ter o **Python 3.x** instalado.
2. Clone o repositório ou baixe o arquivo `imc.py`.
3. No terminal, execute:
   ```bash
   python3 imc.py
4. Digite a sua altura é seu peso quando solicitado (utiliza o `.` ou `,` para separar casas
decimais, ex: `1.73` ou `1,73`.)
